# BottomNavigationBar
Gaussian Blur BottomNavigationBar

这是一个 实时 高斯模糊（类似毛玻璃） 透明的底部任务栏 样式的写法，仅供参考。

原理是：

布局
bottomNavigationBar+BottomAppBar+BottomNavigationBar

高斯模糊：
ClipRect+BackdropFilter+ImageFilter
